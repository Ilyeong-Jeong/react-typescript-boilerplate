# react-typescript-boilerplate
**React와 TypeScript을 base로 한 Boilerplate code**

## Tech Stack
- React v17
- TypeScript v4
- Webpack v5
- React Router v5
- Express v4

## Usage
**Install dependencies**
```
npm install
```

**Build (Webpack Bundling)**
```
npm run build
```

**Run development mode**
```
npm run dev
```

**Run production mode**
```
npm run start
```
